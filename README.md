# FocusFlow

Build a modern, intelligent web application called “TimeScope – Smart Time Tracking & Productivity Analyzer”.

The goal of this application is to reduce manual input and provide semi-automatic tracking of user activities using smart interaction and behavioral insights.

💡 Problem

People feel busy throughout the day but do not understand how their time is actually spent. Manual tracking apps are tedious and users often stop using them.

✅ Solution

This application provides a semi-automated time tracking system that minimizes manual input using timers, smart suggestions, and activity detection.

🔥 Core Features

1. One-Tap Activity Tracking

- Provide predefined buttons such as Study, Phone, Travel, Sleep, Work.
- When a user clicks a button, a timer starts automatically.
- When stopped, the time is recorded without manual entry.

2. Smart Suggestions (Behavior-Based)

- Track user patterns over time.
- Suggest actions like:
  - “You usually study at 7 PM. Start tracking?”
  - “You often use your phone at this time.”

3. Idle Detection (Automatic Tracking)

- Detect user inactivity (no mouse/keyboard interaction).
- Automatically classify idle time.

4. Activity Classification

- Categorize activities into:
  - Productive
  - Non-productive
- Automatically calculate productivity percentage.

5. Daily Analytics Dashboard

- Show:
  - Total time spent per category
  - Productive vs non-productive time
  - Visual charts (pie/bar)

6. Weekly Insights (AI-like)

- Generate insights such as:
  - “You spend 40% of your time on phone usage”
  - “Productivity is lower on weekends”
  - “Your most productive time is in the morning”

🎨 UI Requirements

- Clean, modern dashboard layout
- Card-based UI with gradient background
- Large buttons for quick interaction
- Clear visualization using charts
- Minimal typing required

🛠 Tech Stack

- HTML, CSS, JavaScript
- Firebase (Authentication + Firestore)
- Chart.js for data visualization

🎯 Key Objective

The system should feel “automatic” by minimizing user effort and providing intelligent insights, even though it is built using simple web technologies.

💎 Expected Outcome

A fully functional smart productivity tracking app that demonstrates:

- Semi-automation
- Behavioral analysis
- Real-time insights
- User-friendly interaction
